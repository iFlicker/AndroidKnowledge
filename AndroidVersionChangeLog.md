# Android Version Change Log

> 状态：框架占位，后续补充各 Android 版本的机制变化、API 变化、兼容变化和迁移建议。

## 写作目标
本文件作为 Android 版本演进的独立入口，后续正文应从 Android 早期版本持续记录到 Android 17。它不替代 `docs/5.Android` 下的专题文章，而是按版本维度串联平台变化。

## 记录模板
每个版本后续扩写时建议按以下结构整理：
- 版本定位：该版本的核心主题和平台方向。
- 机制变化：系统机制、运行时、权限、后台限制、UI、存储、安全、性能等变化。
- API 变化：新增、废弃、行为调整和推荐替代方案。
- 兼容变化：targetSdk、权限、隐私、后台任务、存储、通知、安装包等迁移影响。
- 开发者迁移建议：检查清单、适配步骤、测试要点和常见问题。
- 关联文档：链接到 `docs/5.Android` 下的专题正文。
